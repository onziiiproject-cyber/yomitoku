# 9. Claude Code Rules

このプロジェクトでは

必ず

・このデザインガイドを優先する

・Asset Libraryの素材を優先する

・新規イラストは同じ世界観で作成する

・ブランドカラーを守る

・余白を広く使う

・カードデザインを基本とする

・AIを主役にしない

・経営者を主役にする

・機能ではなく価値を伝える

・シンプルで信頼感のあるUIを作る
